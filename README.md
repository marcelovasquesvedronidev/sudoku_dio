# 🧩 Jogo do Sudoku em Java

Este é um projeto prático de um jogo de Sudoku desenvolvido em Java, criado como parte do desafio da plataforma **Digital Innovation One (DIO)**. O projeto exercita conceitos fundamentais de lógica de programação, manipulação de arrays e estruturas de controle, além de oferecer suporte para execução via terminal e interface gráfica.

---

## 🚀 Funcionalidades

- **Lógica de Jogo Consistente:** Validação de regras tradicionais do Sudoku (linhas, colunas e blocos 3x3).
- **Modo Terminal:** Permite jogar diretamente pela linha de comando (`main` branch).
- **Interface Gráfica:** Versão interativa com interface para o usuário (`ui` branch).
- **Inicialização Customizada:** Carregamento do estado do tabuleiro através de argumentos de linha de comando.

---

## 🛠️ Tecnologias Utilizadas

- **Java SE** (Java Development Kit)
- **Git & GitHub** para versionamento
- **IDE Sugerida:** IntelliJ IDEA ou VS Code
